Allophonic transcriptor Beta

A student's aid in learning allophonic transcription

This transcriptor has been created to help you learn allophonic transcription of simple words in:

  • British English (General British)
  • American English (General American)

Use the form below to convert phonemic transcription into narrow phonetic (allophonic) transcription.

For the tool to work properly, you need to input the correct phonemic transcription of a word, e.g. /ɪɡˈzɑːmpᵊl/ will return [ɪɡˈzɑ̃ːmʔpɫ̩] (BrE) or [ɪɡˈzæ̃ːmʔpɫ̩] (AmE). Some processes depend on syllabification. You may add dots <.> to delimit syllabes and improve the quality of the transcription, e.g. /ɪɡˈzɑːm.pᵊl/

This tool works only with single words!
