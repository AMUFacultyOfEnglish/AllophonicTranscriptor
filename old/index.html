Phonetic transcriptor Beta

A student's aid in learning allophonic transcription

This transcriptor has been created to help you learn allophonic transcription of simple words in Received Pronunciation English. Please use the form below to convert phonemic transcription into narrow phonetic (allophonic) transcription.

For the tool to work correctly, you need to input the phonemic transcription of a word, e.g. /ɪɡˈzɑːmpᵊl/ will return [ɪɡˈzɑ̃ːmʔpɫ̩].

This tool will not work with full sentences!
