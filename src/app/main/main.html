Allophonic transcriptor

A student's aid in learning allophonic transcription

This transcriptor has been created to help you learn allophonic transcription of simple words in:

  • British English (Received Pronunciation)
  • American English (General American)

Use the form below to convert phonemic transcription into narrow phonetic (allophonic) transcription.

For the tool to work properly, you need to input the correct phonemic transcription of a word, e.g. /ɪɡˈzɑːmpᵊl/ will return [ɪɡˈzɑ̃ːmʔpɫ̩] (BrE) or [ɪɡˈzæ̃ːmʔpɫ̩] (AmE).

This tool works only with single words!
